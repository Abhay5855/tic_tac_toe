# tic_tac_toe
Tic tac toe game with well GUI. Built using python tkinter library.

### Requirements
python3
tkinter library

### Usage :
```
pip install tkinter
pip install pillow
git clone https://github.com/hitchhicker007/tic_tac_toe.git 
cd tic_tac_toe
python main.py
```
